Welcome! This is the first post on my new blog: My Journey With. I built this site with [Astro](https://astro.build) and deployed it to GitHub Pages.

## Why a blog?

I've been meaning to write more for a while. Having a dedicated place to put thoughts down (and the slight pressure of knowing it's public) seems like a good motivator. The goal is consistency over perfection.

Topics I plan to cover:

- Life
- Music
- Career
- Whatever sparks an idea

## The setup

The site is intentionally simple. Markdown files for content, Astro for static site generation, GitHub Actions for automated deploys. No database, no CMS, no nonsense.

```bash
# Clone and run locally
npm install
npm run dev
```

That's it. Open `http://localhost:4321` and you're off.

## What's next

I'll aim for roughly one post per week, but I won't hold myself to that strictly. Quality over quantity.

Thanks for stopping by.
